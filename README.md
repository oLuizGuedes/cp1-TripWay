# 🌍 TripWay - Agência de Viagens

Projeto desenvolvido para a disciplina de Front-End na FIAP, com o objetivo de criar um site institucional para uma agência de viagens, utilizando HTML e CSS.

O site apresenta destinos turísticos, pacotes de viagem e um formulário de contato, com foco em organização visual, estrutura semântica e responsividade.

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Flexbox
